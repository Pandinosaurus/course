Terminal howto
=======================
是的, 这是一个可以直接输入命令的Python控制台。

-----------------------
它与电脑上的Python控制台无异, 大多数人通常使用它来探索对象的属性, 测试语法或实现他们的想法. 你可以直接输入命令, Python解释器会帮你运行它们. 你还可以:

* 通过点击左上角的加号按钮新建, 或点击下拉列表切换不同的控制台. 要关闭控制台, 只需点击下拉列表中对应窗口的关闭按钮.
* 选择或粘贴文本(通过长按终端显示菜单)

注意
-------
除非您已完全关闭控制台, 否则在通知栏中会有通知. 您可以在任何界面通过点击通知回到已打开的控制台.
